# Imágenes para ProReserve

Este directorio debe contener las siguientes imágenes para la página de inicio:

- `restauracion.jpg` - Imagen para la categoría de Restauración
- `bienestar.jpg` - Imagen para la categoría de Bienestar
- `ocio.jpg` - Imagen para la categoría de Ocio
- `salud.jpg` - Imagen para la categoría de Salud

## Recomendaciones:
- Tamaño recomendado: 640x400 píxeles
- Formato: JPG o PNG
- Peso: Máximo 500KB por imagen para optimizar velocidad de carga
